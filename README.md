# 🔮 Таро · Рандомайзер карт

Веб-приложение на **Vue 3 + Vite**: выбираешь колоду, жмёшь кнопку — выпадает
случайная карта Таро с названием, изображением и трактовкой.

## Возможности

- **Случайная карта** из полной колоды 78 арканов по нажатию кнопки.
- Карта выпадает **прямой или перевёрнутой** случайным образом — трактовка меняется.
- **Две колоды** на выбор, с трактовками на русском по **6 сферам**:
  - 🌞 **Классическая** (Райдер–Уэйт) — любовь, финансы, события по карте,
    совет карты, магическая диагностика, личность и характер.
  - 🌑 **Теневая** (Sola-Busca, 1491, другие изображения) — преграды в отношениях,
    моя тень, скрытые страхи, теневой урок, как прожить тень, психологический портрет.
- **Перетасовать колоду** — отдельная кнопка с анимацией перемешивания.
- **Расклад из 3 карт** — «Прошлое·Настоящее·Будущее» и «Ситуация·Совет·Итог».
- **Поле вопроса** перед раскладом и **история** раскладов (localStorage).
- **Карта дня** — раздел с детерминированной картой на сутки.
- **Энциклопедия** — все 78 карт с фильтром по масти/аркану и поиском.
- **Матрица судьбы** — нумерология по дате рождения с привязкой к старшим арканам.
- Одну и ту же выпавшую карту можно перечитать в другой колоде.
- Название, масть/аркан, цветные ключевые слова, бейдж положения.
- Анимация раскрытия карты, «звёздный» фон, адаптивная вёрстка.
- Работает офлайн: если API недоступно, используются встроенные трактовки.

## Источники данных (открытые)

- **Данные карт:** [tarotapi.dev](https://tarotapi.dev) — открытое API (CORS разрешён).
- **Изображения:** [Wikimedia Commons](https://commons.wikimedia.org), public domain
  (через `Special:FilePath`): классическая — Rider–Waite–Smith, теневая — Sola-Busca.

## Запуск

```bash
npm install      # установка зависимостей (один раз)
npm run dev      # дев-сервер: http://localhost:5174/
```

### Сборка для продакшена / OSPanel

```bash
npm run build    # результат в папке dist/
npm run preview  # локальный предпросмотр сборки
```

Папка `dist/` самодостаточна (`base: './'`) — её можно отдавать любым
статическим сервером, в т.ч. через OSPanel.

## Стек

Vue 3 (`<script setup>`, Composition API) · Vite 5 · нативный CSS (без UI-библиотек).

## Структура

```
src/
├─ App.vue                     # оболочка: навигация по 4 разделам
├─ components/CardDisplay.vue  # карточка: изображение + трактовка по 6 сферам
├─ components/ReadingView.vue  # «Расклад»: 1/3 карты, вопрос, перетасовка, история
├─ components/DailyCard.vue    # «Карта дня» (детерминирована по дате)
├─ components/Encyclopedia.vue # «Энциклопедия»: все 78 карт, фильтр, поиск
├─ components/MatrixView.vue   # «Матрица судьбы»: нумерология по дате рождения
├─ composables/useHistory.js   # история раскладов в localStorage
├─ composables/useTarot.js     # загрузка из API, фолбэк, случайный выбор
├─ data/
│  ├─ decks.js                 # две колоды + сборка трактовки по сферам
│  ├─ classicRu.js             # классические трактовки (5 сфер × 2 положения)
│  ├─ shadowRu.js              # теневые трактовки (5 сфер × 2 положения)
│  ├─ interpretationsRu.js     # русские названия и ключевые слова карт
│  └─ cardImages.js            # URL изображений Wikimedia (RWS + Sola-Busca)
└─ styles/main.css             # глобальные стили / фон
```

> Приложение создано для развлечения.
